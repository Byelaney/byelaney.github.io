三数之和

给出一个有n个整数的数组S，在S中找到三个整数a, b, c，找到所有使得a + b + c = 0的三元组。在三元组(a, b, c)，要求a <= b <= c。

结果不能包含重复的三元组。

思路和二数之和很像，这里关键是对不重复的处理，我的处理比较粗糙写的比较丑陋。
